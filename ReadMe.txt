Пронграмма парсит любой вложенный файл с соблюдением JSON-структуры.

Доступные команды для получения информации о перелетах:
    min - минимальная стоимость перелетов
    max - максимальная стоимость перелетов
    avg - средняя стоимость перелетов
    exit - выход из программы

Доступна компиляция в исполняемый jar-файл.
